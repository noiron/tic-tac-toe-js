利用 Minimax 算法和 Alpha-Beta 剪枝算法来实现井字棋（Tic Tac Toe）的 AI 功能。

以下链接为使用 Alpha-Beta 剪枝算法的展示：
https://tic-tac-toe-js-alpha.vercel.app/

本地运行：

    git clone https://github.com/noiron/tic-tac-toe-js.git
    cd tic-tac-toe-js
    npm i
    npm run dev
    // open http://localhost:1234
